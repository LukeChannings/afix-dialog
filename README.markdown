# Dialog

A Web Component for presenting a modal dialog

## Using it

Try it out directly (requires a modern browser)

```javascript
import "https://cdn.skypack.dev/afix-dialog@latest";
```

Or install with

```bash
npm install afix-dialog
```

And use like this

```js
import "afix-dialog"`
```
